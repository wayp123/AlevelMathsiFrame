Integration Involving Linear Substitution(0001).

Currently we can not integrate directly, a composite function, if we need to integrate a composite function, we need to perform a linear substitution(0002).

To integrate the complex function(0003).

 

 fgx, gx must be a linear function,

we need to use the following procedure(0004).

 

1,   Set u = g x(0005).

2, Find the derivative of u, {d u dx} and solve for dx_(0006).

3, In the function replace g x, with u, and dx with the result obtained in step 2(0007).

4, Integrate the function with respect to u(0008).

5, Replace u, with g x_(0009).
