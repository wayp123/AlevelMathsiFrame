The Trigonometric Ratios Of An Angle(0002).

 

We use the triangle, in the diagram, to define, the three basic, trigonometric ratios, of the triangle, side lengths, using angle, A(0003). A good mnemonic is, the acronym, SOHCAHTOA, Sine, Opposite Hypotenuse, Cosine, Adjacent Hypotenuse, Tangent, Opposite Adjacent(0004). Remember, if you are using a calculator, to obtain the value, of a trigonometric ratio, make sure, that it is in the proper mode, it should be in radian mode, if the angle is in radians, and degree mode, if the angle is in degrees(0005). You can find the angle, that corresponds to a value, using the inverse, of each function, usually listed as, cos, minus 1, sin, minus 1,tan, minus 1, on your calculator(0006). A formal discussion, of the inverse trigonometric functions, will be in Core 3(0007). The vertical blue, dashed lines, in the tangent graph, are the asymptotes, of the tangent function(0008). The tangent function, will not be defined, at these points, because at these points, the cosine graph, is zero, see the tangent identity(0009).
