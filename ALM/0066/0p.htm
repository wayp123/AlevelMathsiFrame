The Modulus Function, of real numbers(0002).

For any real number A, the absolute value, or modulus of A, is denoted by mod A, _a vertical bar, on each side of the quantity_ and is defined as(0003).

mod A, = A, if A, is greater than or = 0, and mod A,= negative A, if A is less than 0(0004). Note that when A, is less than 0, then, a negative by negative, is a positive number(0005).

As can be seen from the above definition, the absolute value of A, is always either, positive, or zero, but never negative(0006).

Solving Modulus Inequalities(0013).

When solving an inequality involving a modulus, it is important to remember, that you cannot simply ignore the modulus(0014).

Instead you need to use the rule, to remove the modulus(0015).

if mod x, is less than b, then, negative b, is less than x, and x, is less than b(0016). If x = neg 1, and b=2, we can see that neg 1, is greater than neg 2, and mod x = 1, is less than 2(0017).

Then you solve as you would a regular problem, In example(0018).
