Finding A Root Using a Graph(0006).

 

In the graph we have two functions(0007). If we want to approximate, where the roots, are, we have to look at where the function, crosses the x-axis(0008). The lilac function, crosses the x-axis, somewhere between 2.05, and 2.25(0009). The green function, has roots around 1, and 3.5(0010). If we want to know where the green function, is equivalent to the lilac function, we need to look at the graph(0011). When the two graphs cross, they are equivalent(0012). This number will be the root(0013). In this case it is around 1.75(0014). We can also say, that on this domain, the functions will only cross once(0015).

Finding A Root, By Searching for A Sign Change(0016).

 

When a function has a root, the value of the function will change, from a positive value, to a negative value, or vice versa(0017). See the table, of values, for the lilac function, we can say that the root occurs, between 2.05, and 2.10(0018).

Iterative Formulae(0019).

 

An iterative formula, is a formula, that is composed of itself(0020). That is, the output of the function, is the next input, of the function,

x, n plus 1, = F n(0021).

 

These functions, are a sequence of approximations, that usually converge, to the value of a function(0022). You can tell if a function converges, by the fact that the outputs, become closer and closer to each other, if this does not happen, then the function diverges, and there is no value(0023).

 

b0005

 

The output of an iterative formula is written as(0024).

The first x is provided for you(0025). This value can be obtained from the graph(0026). The output of the iterative formula is written as the Greek letter alpha(0027). Alpha can be found to the required degree of accuracy when,  x n, = x n plus 1, to the required number of decimal places(0028). That is, the previous value = the next value, at the required decimal(0029). Given an iterative function, you can find the root of an equation(0030). Also you find the function, from a given iterative function by setting, x = iterative function, then solving, so that you have a zero on one side(0031). The aforementioned process can be done in reverse(0032). Functions that have fractional indexes, are more likely to converge(0033).
