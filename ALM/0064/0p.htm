Basic Properties(0026).

A function, relates, each of its inputs, to exactly one output(0027). If there are, two outputs, the formula, is not a function(0028). For example_, y = x squared, is a function, because every x, has only one output, but  y squared = x, is not a function, because every x, can have two values(0029). For example 4 = negative 2, squared, and  2, squared, so x can be, 2, or negative 2(0030). Graphs of circles _and similar shapes_ represent many, to many, mappings, and so they are not functions(0031). You can determine, if a graph is a function, by using, the vertical line test(0032). This test states, that a graph, is a function, if and only if, any vertical line drawn, will intersect the graph, only once(0033).

There are four ways, to represent a function:(0034).

1_ A Formula(0035).

2_ A Graph(0036).

3_ A Table(0037).

4_ Verbally(0038).

Domain and Range(0039).

Every function, has a domain, and range(0040). The domain, is the set of the input values(0041). The range, is the set of the output values(0042). Remember, if there is a reflection, or translation, in the function, there will be a reflection, or translation, in either one, or both, the domain, and range(0043).

In example,:

The domain of_,  f x, = positive root x, is 0 to infinity, and the range is  0 to infinity_(0044). but the domain of, f x, = negative root, x plus 6, is,  negative 6, to infinity, and the range is,  0 to negative infinity(0045).

 

Note: a function can only have a positive root, or negative root, formula, but not both(0046).

 

See: Reflection Rule 1, and Translation Rule 2, in OCR C1(0047).

One-One Function_(0065).

A one-one function, must have a unique output, for every input(0066). In example, f x = x2, will not be one-one because the inputs, -2 and 2, will both produce the output 4, but f x = x3, will be one-one because every input, will produce a unique output, f -2 = -8, and f 2 =8(0067). You can determine if a graph is one-one, by using the horizontal line test(0068). This test states, that a function is one-one, if and only if, any horizontal line drawn, will intersect the graph only once(0069).
