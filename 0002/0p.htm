Indices and Surds8(0001). x by 3 is a shortening

The indices in this chapter heading, refers to the power, to which a number is raised(0002). Thus, x squared is a number with an index of 2(0003). This is never normally said, as people prefer the phrase, x to the power of 2"(0004). Likewise, surds is a heading that is not often used(0005). Occasionally you will be asked to give an answer in surd form; this implies that you should give the answer in terms of constants, trig funcions, and square roots, instead of working out an imprecise decimal approximation(0006). 

b0002 

Indices_(0007). 

As you should know from G-C-S-E,

x by 3 is a shortening of x times x times x(0008). In the same way, any number to the power of n, is that number multiplied by itself n times(0009). To describe in more detail, in the expression x by 3, the x is referred to as the base, and the 3 as the exponent(0010). 

By always reducing the expression back to its basic parts, it is possible to come to conclusions about how to treat numbers raised to powers in algebra(0011).

b0003 

Multiplication(0012). 

When you multiply indexed expressions _indices_, you add the exponents together(0013). 

x by 3 times x by2 = x times x times x times x times x = x by 5(0014). 

As you can see,

x by 3 times x by 2 = x by 3 plus 2 = x by 5(0015).

b0004 

Division, _fractions_(0016). 

Division is, as expected, the opposite to multiplication(0017). When you divide indices, you subtract the exponents from each other(0018).

"Again we can show that,

x by 4 over, x by 2 = x by 4 minus 2 = x by 2(0019). 

b0005 

Negative powers(0020). 

Having done the above division we begin to ask ourselves what happens when we have a fraction that has a higher exponent on the bottom than the top(0021). 

Using the trusted method of showing each x, seperately, we obtain:(0022). 

x by 2 over, x by 3 = 1 over, x(0023).

b0006 

This just demonstrated the above method of subtracting the exponents:(0024). 

x by 2 over, x by 3 = x by, 2 minus 3 = x by neg 1(0025). 

This implies that x to the power of a negative number, is one divided by x, to the power of that number(0026).

The specific case of  1 over, x, or  x by neg 1, is referred to as the reciprocal of x, or more often, as the inverse, of x"(0027).

b0007 

Base raised to two powers_(0028). 

When you have a base raised to two powers, you multiply the powers(0029). 

As you can see that",

x by 2, by 3 = x by 2, times 3 = x by 6(0030).

b0008 

Multiple bases to the same power(0031). 

When you have two bases to the same power, you can raise both bases to the same power, and multiply them(0032). 

For example,

x y, by 3=x y times x y times x y = x times y, times x times y, times x times y, = x times x times x, times y times y times y(0033). 

is the same as

x by 3, y by 3(0034). 

b0009

There is a similar situation with division,

x over, y, by 2 = x over, y, times x over, y, = x times x, over, y times y, = x by 2 over, y by 2 = x by 2, y by neg 2(0035).

b0010 

Fractional powers(0036). 

What if the power, or index isn t even an integer but a fraction?(0037). Suppose you wanted to find  x by 1 over, 2(0038). 

You can say that,

 

x by half times x by half = x by 1 _by index multiplication law_(0039).  

b0011

Which means that,

 x by half, must be the same as  the square root of x(0040). 

However it is customary to only use the positive root, and so(0041).

 x by half is defined as the positive square root of x(0042). You can use a similar argument for other such fractions, for example(0043). 

x by 1 over, 3 by 3 = x(0044). 

b0012

so,

x by 1 over, 3 = the 3rd root of x(0045). 

In cases when the numerator is not 1, we need to use other laws of indicies, to prove the square definition for example(0046). 

x by 2 over, 3 = x by 2, by 1 over, 3 _law, base, raised to 2 powers_"(0047). 

b0013

we now know that",

x by 2, by 1 over, 3 = 3 root of, x by 2(0048). 

therefore,

x by A, over, b = b root of, x by A, is a useful general rule(0049).

b0014 

The 0, and 1 Power(0050). 

You may well have realized that,

x by 1 = x(0051). 

You can prove this, because we know that,

x by n over, x by n minus 1, = x(0052). 

'which is equivalent to,

x by n, minus n minus 1, = x by 1, = x, _using the division law_(0053). 

b0015

Similary with  x by 0 we can prove that it is equivalent to 1(0054). 

Since,

x by n, over, x by n, = 1(0055).

'which is equivalent to,

x by n, minus n, = x by 0 = 1(0056).

b0016 

The Laws of Indices, or exponents(0057). 

The rules that have been suggested above are known as the laws of indices, and can be written as:(0058). 

x by A times x by b = x by A plus b(0059). 

x by A over, x by b = x by A minus b(0060). 

x by neg n, = 1 over, x by n(0061).

b0017 

x by A by b = x by A b(0062). 

x y by n, = x by n, y by n(0063). 

x over, y, by n, = x by n, over, y by n(0064). 

x by A over, b = the b root of x by A(0065). 

x raised by 0 = 1(0066). 

x raised by 1 = x(0067).
